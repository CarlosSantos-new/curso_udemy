Lista Não Oredenada

Para poder criar uma lista ordenada, usa se, a tag (ul). Para poder adcionar linha no mesma tenho que usa a tag (li)

O uso dessa formatação tem como objetivo de marca o inicio do para usando o type como (circle, square, disc).

Exemplo de lista Não ordenada

  • Home
  • Quem Somos
  • Contato
    •

Já na lista Ordenada tem uma sequência para se iníciar, para construr, temo que usa tag (ol), como na lista não ordenada, para criar linha, temos que usar tag (li). O type da lista ordena são (1, I, a, A).

Exemplo de lista Ordenada

  1. Espiritual
  2. Emocional
  3. Família
  4. Profissional
